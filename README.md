# Valley of Monument
Dit is een recreatie van de game monument valley. Monument Valley is een puzzelspel waarin je door bijzondere gebouwen loopt. Je draait platforms, schuift trappen en gebruikt optische illusies om nieuwe paden te maken.

Hierbij hebben we onze eigen spin aan gedaan. onze thema is een sweeuige omgevin met steampunk elementen In de art style van  arcane.
